치매 예측 분류 모델 만들기

OASIS Longitudinal MRI Dataset — 실습

📁 336 observations 📊 15 variables 🎯 Binary Classification
// 분석 목표

MRI 및 임상 데이터로 치매 여부를 예측할 수 있을까?

워싱턴 의대 OASIS 프로젝트의 종단 MRI 데이터를 활용하여 로지스틱 회귀 분류 모델을 구축합니다.
나이, 인지 검사 점수, 뇌 부피 등의 임상 지표로 치매(Demented) / 비치매(Nondemented)를 예측하고, 모델의 성능을 평가하며 치매 진단에 가장 큰 영향을 미치는 요인을 파악합니다.

Dataset
OASIS-2 Longitudinal
대상
60~96세 노인 150명, 373 MRI 세션
종속변수
Group (Demented / Nondemented)
Data URL
https://raw.githubusercontent.com/ruminoDesign/source_data/refs/heads/main/dementiaDataSet.csv
💡 데이터 로드 방법: 위 URL을 pd.read_csv(url) 에 넣으면 바로 불러올 수 있습니다. 생성형 AI를 활용해 각 단계의 코드를 직접 작성해 보세요.
STEPS

분석 과정

아래 5단계를 순서대로 수행하세요. 코드는 직접 작성합니다.

01
데이터 로드 및 탐색
  • URL에서 데이터를 불러오고 처음 5행을 출력하세요.
  • 전체 행·열 수, 변수 타입을 확인하세요.
  • 각 변수의 기술통계량(평균, 표준편차, 최솟값, 최댓값)을 확인하세요.
  • 결측값이 어느 변수에 몇 개 있는지 파악하세요.
  • 종속변수 Group 의 클래스 분포(치매 vs 비치매 비율)를 확인하세요.
02
데이터 전처리
  • 분석에 불필요한 변수(Subject.ID, MRI.ID, Hand)를 제거하세요.
  • 종속변수 Group 을 Demented = 1, Nondemented = 0 으로 변환하세요.
  • 범주형 변수 gender 를 숫자(0/1)로 인코딩하세요.
  • 결측값을 적절한 방법으로 처리하세요. (평균 대체 또는 행 제거 — 둘 다 시도해 보세요.)
  • 데이터를 훈련 80% / 테스트 20% 로 분할하세요.
03
로지스틱 회귀 모델 학습 및 예측
  • 훈련 데이터로 로지스틱 회귀 모델을 학습시키세요.
  • 테스트 데이터에 대한 예측 클래스(0 또는 1)와 예측 확률을 각각 구하세요.
04
모델 평가 — 혼동행렬 & ROC 커브
  • 혼동행렬(Confusion Matrix) 을 시각화하고, TP·FP·FN·TN을 확인하세요.
  • 정확도(Accuracy), 정밀도(Precision), 재현율(Recall), F1 점수를 출력하세요.
  • ROC 커브를 그리고 AUC 값을 구하세요.
  • 결과를 해석하세요. 이 모델은 치매를 잘 예측하나요?
05
치매 진단에 가장 큰 영향을 미치는 요인은?
  • 각 변수의 회귀계수(β)와 오즈비(Odds Ratio = exp(β)) 를 계산하세요.
  • 오즈비를 막대그래프로 시각화하세요. (OR > 1: 위험 증가 / OR < 1: 위험 감소)
  • 치매 위험을 가장 높이는 변수와 가장 낮추는 변수는 무엇인가요?
  • 결과를 생명과학적 관점에서 해석해 보세요. 예상과 일치하나요?
⚠️ 주의: CDR(치매 등급)은 Group과 직접적으로 연관된 변수입니다. CDR을 독립변수로 포함하면 모델 성능이 인위적으로 높아질 수 있어요. CDR을 포함했을 때와 제외했을 때 결과를 비교해 보세요.
